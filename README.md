# 🙇🏻 Java_Tetris
Java 테트리스 게임

## 프로젝트 설치 및 실행
1. IDE 준비 (Eclipse 설치)
> 해당 [링크](https://www.eclipse.org/downloads/)로 들어가서 다운로드 하세요.

2. Repository 복제
- 소스 파일 설치
```shell
$ git clone https://github.com/iamsoojung/Java_Tetris.git
```
 
3. 프로그램 실행
- jar 파일 실행
```shell
$ java -jar .\java_tetris.jar
```
